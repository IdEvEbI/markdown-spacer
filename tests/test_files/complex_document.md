# 项目标题

这是一个包含中英文混排的文档。

## 功能列表

- 支持中文English混排
- 支持数字123处理
- 支持数学符号A+B运算
- 支持嵌套列表：
  - 子项1English
  - 子项2中文
  - 子项3数字123

## 有序列表示例
